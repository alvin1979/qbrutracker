1 ОПИСАНИЕ

QBRutracker это поисковый плагин для QBittorrent с открытым исходным кодом, написанный на Python 3, предоставляющий удобный способ поиска по оффлайн базе торрентов сайта Rutracker.org.

Особенности:
	- Быстрый и надежный поиск торрентов работающий локально и независимо от доступности Rutracker.org;
	- Компактность (новейшая база торрентов в bzip занимает 90MB);
	- Работает на всех платформах;
	- Исходные коды открыты и доступны по лицензии GPLv3;
	- Настраиваемый список трекеров;
	- Удобный поиск не выходя из клиента (поиск по категории, упорядочивание по размеру, исключающие слова, ограничение выдачи);

Системные требования:
	- QBittorrent qbittorrent.org
	- Python 3 python.org
	
	
2 ИСПОЛЬЗОВАНИЕ
	
Чтобы установить...
	1 скачать qbrutracker.py и следуйте инструкциям http://www.techsupportalert.com/qbittorrent-help-torrent-search-engine 
	2 скачать magnet:?xt=urn:btih:628F28C998A79D780455DC1B60892DD721EE44D1 (или http://rutracker.org/forum/viewtopic.php?t=4824458) в папку %localappdata%\qbittorrent
	3 [необязательно] разместите список трекеров "tracker.txt" в папке %localappdata%\qbittorrent\rutracker-torrents

Во время поиска...
	будут найдены все торренты, описание которых содержит все веденные в строку поиска слова. Исключаются слова, начинающиеся с символа "-". Количество найденных торрентов ограничено заданным лимитом, обозначенным символом "=" за которым следует цифровое значение. Пример: "фотографии москва -встреча =10", найдет максимум 10 торрентов содержащих слова "фотографии" и "москва", но не содержащие слово "встреча".

Чтобы упаковать базу данных...
	введите в строку поиска ключевое слово QB_PACK, это запустит процесс упаковки базы данных (папка rutracker-torrents будет упакована в архив под именем rutracker-torrents.tar.bz2, затем папка rutracker-torrents будет удалена), что существенно уменьшит занимаемое место, но в свою очередь замедлит процесс поиска по базе.
